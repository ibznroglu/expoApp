## Project Overview

The expoApp is a React Native trivia/quiz game application built with Expo and Expo Router. The project implements:
- **Framework:** Expo with React Native
- **Routing:** Expo Router (file-based routing in /app directory)
- **Authentication:** Custom AuthContext with Appwrite backend
- **Styling:** StyleSheet.create pattern with three dedicated style files
- **Components:** Two reusable components (TextCustom, ListItem)
- **Screens:** Home/index, Quick Game, Sign In, Sign Up, Verify Email, Forgot Password, Reset Password

The application has a **dark blue theme with orange accents** primarily used for game UI elements, along with green (success), red (error), and purple (tournament) highlight colors.
